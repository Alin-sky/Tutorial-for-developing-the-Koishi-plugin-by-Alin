    - **TypeScript是什么呢**
        -  TypeScript 是 JavaScript 的一个超集
        -  TypeScript 扩展了 JavaScript 的语法
        -  TypeScript 可处理已有的 JavaScript 代码，并只对其中的 TypeScript 代码进行编译
        -  TypeScript 的类型定义相较于 JavaScript 显得更为严格和精确。
